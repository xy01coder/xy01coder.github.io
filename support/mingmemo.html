Frequently Asked Questions

üé§ Recording Issues

Problem: Cannot record voice

Solution: Make sure microphone permission is enabled: Settings → Ming Memo → Microphone. Also ensure no other app is using the microphone.

üìù Transcription Issues

Problem: Voice-to-text is inaccurate or fails

Solution: Ensure you have a stable internet connection. AI transcription requires network access. Record in a quiet environment and speak clearly for better accuracy.

üíæ Data Issues

Problem: Memos are lost or cannot be saved

Solution: Ensure your device has sufficient storage space. Try restarting the app. If the issue persists, restart your device.

üì§ Export Issues

Problem: Cannot export Excel file

Solution: Ensure your device has enough storage space. After export, use the share function to send the file to other apps or devices.

System Requirements

  • iOS: 15.0 or later
  • Permissions: Microphone (required for voice input)
  • Network: Internet connection required for voice transcription
  • Storage: Minimal storage space required

How to Use

Voice Input

  1. Long press the "+" button on the home screen to start recording
  2. Speak your memo content
  3. Release to stop recording and begin AI transcription
  4. Edit the transcribed content on the edit screen
  5. Tap save to create the memo

Manual Input

  1. Tap the "+" button on the home screen
  2. Type your memo content directly
  3. Optionally set a time and priority
  4. Tap save to create the memo

Managing Memos

  • Tap a memo to open the edit screen
  • Check the checkbox to mark a memo as completed
  • Use the left drawer menu to filter by time range (Today/This Week/This Month/All)
  • Use the top-right menu to switch sort order (by time/priority)
  • Use the search bar to quickly find memos

Data Export

  • Tap the top-right menu and select "Data Export"
  • Choose the export range
  • Export as Excel and share via the share function

Troubleshooting Steps

  1. Restart the app: Force quit and reopen Ming Memo
  2. Check permissions: Confirm microphone permission is granted in Settings → Ming Memo
  3. Check network: Voice transcription requires an internet connection
  4. Update the app: Make sure you have the latest version installed
  5. Restart device: Sometimes a device restart can resolve issues
  6. Reinstall: As a last resort, uninstall and reinstall (Note: this will delete all memo data)
